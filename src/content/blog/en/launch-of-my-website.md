Alright, here we go 🚀. First real post on this brand new blog, so let's start at the start: **this site is, quite obviously, vibe-coded with AI**. Yeah, because I haven't done frontend in a long time, so I might as well delegate it and focus on what I'm actually good at (**backend / AI / data / architecture / project management**).

For the record, the site is **static** and runs on **Astro**, **Tailwind v4**, i18n FR/EN, and was vibe-coded with **Claude Code**, Anthropic's command-line tool. A few things, like the KPIs, tags, and featured projects, get recomputed at build time — less maintenance on my end 🙌. All I have to do to add a post or a new experience is add/edit a markdown (.md) or yaml file.

I know what some of you are already thinking: "great, another site slapped together by an AI, it's going to smell like a reheated Bootstrap template" 🥐. Except no — the real added value here is the **visibility** it gives me, especially through this blog, where I plan to write about IT topics I find promising and that I'm genuinely passionate about!

*So welcome aboard, and see you very soon for the rest of the adventure! 😃*
